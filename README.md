# 34金币时间管理系统

基于努尔艾力·阿不利孜提出的34金币时间管理法的Web应用，支持移动端安装使用。

## 功能特点

- **时间金币记录**：将每天7:00-24:00的17小时划分为34个"时间金币"，支持记录每个金币的使用情况
- **五维时间评估**：使用蓝绿黄橙红五种颜色标识不同质量的时间使用
- **数据可视化**：提供饼图、柱状图和折线图等多种统计图表
- **智能分析**：自动识别"金币消费陷阱"并提供改进建议
- **离线使用**：支持PWA功能，可安装到移动设备上离线使用
- **数据导入导出**：支持备份和恢复时间记录数据

## 技术栈

- Vue.js 3 + Vite
- Pinia（状态管理）
- Vue Router（路由）
- Chart.js（数据可视化）
- PWA（渐进式Web应用）

## 如何使用

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 部署

1. 将构建后的`dist`目录部署到任何静态文件服务器
2. 由于实现了PWA功能，用户可以在访问网站时选择"添加到主屏幕"来将其作为应用安装到移动设备上

### 移动设备使用

1. 在移动设备的浏览器中访问部署的网站
2. 浏览器会提示安装应用，或者手动在浏览器菜单中选择"添加到主屏幕"
3. 应用将安装到设备上，可离线使用

## 时间金币分类

- **蓝色**：高质量的休闲时间
- **绿色**：基本生理需求时间
- **黄色**：有价值的创造时间
- **橙色**：被动消费时间
- **红色**：无效时间

## 项目说明

本项目基于34金币时间管理法，帮助用户记录、分析和改善时间使用效率。通过将时间可视化和分类，让用户更直观地了解自己的时间分配情况，从而做出更合理的时间规划。
